# 本宮市 消防水利マップ（消火栓・防火水槽 点検／訓練管理）

単一のHTMLファイルで動く、本宮市消防団向けの消防水利マップです。消火栓・防火水槽の位置と点検状況の管理、水利空白（火災リスク）の可視化、送水訓練の計画・計算、訓練計画のPDF出力までを、スマホ1台で行えます。データは各自の端末内に自動保存されます。

---

## 収録ファイル

- `index.html` … アプリ本体（これ1つで動きます）
- `data.json` … 共有データ（任意。あれば初回起動時に自動で読み込みます）
- `data.example.json` … 共有データのテンプレート
- `README.md` … このファイル

---

## 起動方法

- `index.html` をブラウザで開くだけで動作します（ダブルクリック、またはWebサーバー上に置いてURLで開く）。
- **初回はインターネット接続が必要**です（地図タイルと地図ライブラリを読み込みます）。一度開けば表示設定やデータは端末内に保存されます。
- スマホ・PCの主要ブラウザ（Safari / Chrome / Firefox / Edge）に対応。ノッチ端末のセーフエリアや、iOS版Firefoxの表示高さにも対応済みです。

## ヘッダーのボタン

| ボタン | 内容 |
|---|---|
| ☰ 一覧 | 地点リスト・絞り込みの開閉（スマホ） |
| ＋ 地点を追加 | 地図をタップして消火栓・防火水槽を登録 |
| 🎓 訓練 | 送水訓練の計画（水元・火点・中継・屈曲・メモ） |
| 📍 現在地 | GPSで現在地へ地図を移動 |
| ⛶ 全画面 | 全画面表示に切替（対応ブラウザのみ表示） |
| ❓ ヘルプ | アイコンの説明・操作方法 |
| ⋯ データ | バックアップの書き出し・読み込み |

※ アプリ内の「❓ヘルプ」にも同じ説明と基本操作を掲載しています。

## 基本の使い方

### 地点の登録・編集
- **追加**：「＋地点を追加」→地図をタップ→編集パネルで種別・名称・地区・状態・メモなどを入力して保存。
- **編集**：マーカー、または一覧の項目をタップ→編集パネルが開きます。
- **写真**：編集パネルの「写真を追加」から**撮影・ファイル選択のどちらも**選べます（複数可、保存時に自動で圧縮）。
- **座標**：緯度経度は直接入力のほか、「📍現在地の座標を取り込む」でGPSから取得できます。
- **半径**：地点ごとに「水利基準円」「ホース展張円」の半径を個別設定できます（未入力は全体設定を使用）。

### マーカーの見かた
- ピンの色＝種別：🔴 消火栓（赤）／🔵 防火水槽（青）
- 足元の丸＝状態：🟢 正常 ／ 🟠 要点検 ／ 🔴 破損 ／ ⚪ 未確認

### 絞り込み・検索
- サイドバーの種別・状態チップ、地区の選択（＋「移動」でその地区へ）、名称・メモのキーワード検索。

### 表示レイヤー（地図右上）
- 背景：標準地図／淡色地図／航空写真（地理院タイル）の切替
- **水利基準円**（100/120/140m）、**ホース展張円**（半径可変）
- **水利空白（火災リスク）**：基準からの距離に応じて色分け表示（破損した水利は除外して判定）

## 🎓 送水訓練シミュレーション

「🎓訓練」を押すとツールバーが表示され、地図上に送水計画を作れます。

- **💧水元 → 🔥火点** の順にタップして配置（水元は近くの既設水利に自動スナップ）。
- **➕中継**：中継ポンプを置くと区間が分割され、ポンプ台数が増えます。
- **📐屈曲**：送水ラインを**道路に沿って曲げる**中間点です。**ポンプ台数は増えず**、折れ線の**実延長**でホース本数を計算します。タップ地点はラインの一番近い区間へ自動挿入されるので、道なりに何点でも追加できます。
- **📝メモ**：地図上に注記を配置（位置・縦横・文字サイズ・太字を調整可）。
- 必要な**ホース本数**・**可搬ポンプ台数**・**区間ごとの距離**を自動計算し、中継目安を超える区間は⚠で表示します。ホース1本の長さ・余裕係数・中継目安はカードで変更できます。
- 各マーカーはドラッグで移動、吹き出しやカードの一覧から削除できます。

### 訓練計画の保存・振り返り
- カードの「🗂 計画を保存・履歴」から、**実施日・地区・名称**を付けて計画を保存できます。
- 保存済みの計画は「開く／上書き／削除」ができ、日付ごとに振り返れます。

### PDF出力（A4 1ページ）
- 各計画、または作業中の計画を **PDFとして保存／印刷**できます。
- PDFには **訓練範囲の地図（画像）**、送水系統図、送水計算（区間ごとの距離・本数）、メモが**A4 1ページ**に収まる形で出力されます。
- 地図の背景は、表示レイヤーで選んでいる**標準地図／航空写真**に追従します。

## データの保存とバックアップ

データ（地点・訓練計画・表示設定）は**各自の端末（ブラウザ）内**に自動保存されます。「⋯データ」から次の操作ができます。

- **JSONで書き出し**：写真も含めて丸ごと保存。復元や機種変更の引き継ぎに。
- **JSONを読み込み**：書き出したファイルから復元（既存データとの統合も可）。
- **CSVで書き出し**：Excel等で開ける点検一覧（写真は含みません）。
- **共有データ(data.json)を再読込**：配布された最新の共有データで表示を更新。
- **全データを削除**：この端末の全地点を消去（元に戻せません）。

> 大事な記録は「JSON書き出し」で定期的にバックアップしてください。ブラウザのデータ消去や別ブラウザの利用で、端末内保存は失われることがあります。

## 共有データ（data.json）

分団全員に「同じ消火栓・防火水槽の一覧」を初期表示させたい場合は、`index.html` と**同じ場所**に `data.json` を置きます。

1. 代表者が現地調査してアプリに入力
2. 「⋯データ」→「JSONで書き出し」でファイルを保存
3. ファイル名を **`data.json`** に変更
4. `index.html` と同じフォルダ（同じ場所）に配置
   - テンプレートは `data.example.json` を参照

### 自動読み込みのルール
- **端末内にまだ何も保存していない人**（初回・新しい端末）… 開くと `data.json` の内容が自動で読み込まれます。
- **端末で追加・編集・削除した人**… その端末では「自分の状態」が優先され、`data.json` は自動では読み込まれません。
  - 特に、**「全データを削除」した後は、開き直しても `data.json` を自動読み込みしません**（消したデータが勝手に復活しません）。
  - 共有データを取り込みたいときは「⋯データ」→「共有データ(data.json)を再読込」を実行してください。
- 内容を更新したいときは、新しい `data.json` を同じ場所に上書きしてください。

※ `data.json` はローカルでファイルを直接開いた場合（`file://`）は取得できないことがあります。共有データを使う場合はWebサーバー上に置いて開いてください。

## 取り扱いの注意

- 消火栓・防火水槽の位置や破損状況、写真などは、**公開範囲に注意**してください。`data.json` を誰でもアクセスできる場所に置くと、その内容も閲覧可能になります。分団・本部で取り扱いを確認のうえ運用してください。
- 個人が写る写真や、住居が特定される情報の扱いにも配慮してください。

## よくある質問

- **地図が出ない／真っ白** … 初回はインターネット接続が必要です。電波のある場所で開いてください。
- **カメラしか起動しない** … 「写真を追加」はOSの選択画面から撮影・ライブラリ・ファイルを選べます（端末の設定にもよります）。
- **データが消えた** … ブラウザのデータ消去や別ブラウザ利用で端末内保存が失われることがあります。「JSON書き出し」で定期バックアップを。
- **みんなの入力をまとめたい** … 各自の「JSON書き出し」を代表者が集約・統合し、`data.json` を更新する運用が簡単です。
